# 🎬 Netflix Clone – Premium Streaming UI

A fully responsive, high-fidelity Netflix-inspired landing page built using modern frontend technologies. This project focuses on replicating Netflix’s premium UI/UX with pixel-perfect design, smooth interactions, and responsive layouts.

---

## 🚀 Live Demo  
👉 https://project-nishantfix.netlify.app/

---

## ✨ Features

- 🎥 **Interactive Media Sections**  
  Embedded auto-playing videos inside device frames with perfect alignment.

- 🌐 **Multi-Language Support**  
  Toggle between English and Hindi (headings, buttons, inputs).

- 📱 **Fully Responsive Design**  
  Works seamlessly across all devices (mobile → desktop).

- 🎞️ **Trending Movies Row**  
  Horizontal scroll with styled ranking numbers.

- ❓ **FAQ Accordion**  
  Smooth expandable/collapsible sections.

- 🧩 **Custom UI Components**  
  Sticky navbar, modern forms, and scroll effects.

---

## 🛠️ Tech Stack

- **HTML5** – Semantic structure  
- **CSS3** – Flexbox, Grid, animations, variables  
- **JavaScript (Vanilla)** – DOM manipulation, interactivity  

⚡ No frameworks used → performance-focused build

Open `index.html` in your browser
